## 서비스 개요

**바이브폴리오(Vibefolio)** 는 크리에이터와 디자이너를 위한 포트폴리오 플랫폼이에요.

- 주소: https://vibefolio.net
- 무료로 사용 가능
- 회원가입 후 바로 프로젝트를 올릴 수 있어요

### 핵심 기능
1. **Quick Post** — URL만 입력하면 AI가 자동으로 소개글, 썸네일, 카테고리를 생성
2. **V-Audit** — 전문가에게 체계적인 피드백을 받을 수 있는 평가 시스템
3. **에피소드(버전 관리)** — 프로젝트 업데이트를 타임라인으로 관리
4. **AI 도구** — 페르소나 빌더, 린캔버스, 기회 매칭 등

## 포트폴리오 만드는 방법

### 방법 1: Quick Post (추천)
1. vibefolio.net/project/quick-post 접속
2. 프로젝트 URL 입력 (배포된 사이트 또는 GitHub 링크)
3. AI가 자동 분석: 제목, 소개글, 썸네일, 기술 스택, 카테고리 추천
4. 내용 확인/수정 후 게시

### 방법 2: 직접 작성
1. vibefolio.net/project/upload 접속
2. 리치 에디터로 자유롭게 작성
3. 커버 이미지 업로드
4. 장르/분야 선택 후 게시

### 장르 카테고리 (택 1)
포토, 웹툰/애니, 그래픽, 디자인, 영상, 영화·드라마, 오디오, 3D, 텍스트, 코드, 웹/앱, 게임

### 분야 태그 (최대 3개)
경제/금융, 헬스케어, 뷰티/패션, 반려, F&B, 여행/레저, 교육, IT, 라이프스타일, 비즈니스, 마케팅, 문화/예술, 기타

## 템플릿/디자인

vibefolio.net/templates 에서 15개 이상의 템플릿을 제공해요.

### 템플릿 카테고리
- **랜딩페이지** — 서비스 소개, 앱 소개용
- **이커머스** — 쇼핑몰, 상품 소개
- **비즈니스** — 기업/스타트업 소개
- **교육** — 강의, 학습 플랫폼
- **F&B** — 식음료, 레스토랑
- **라이프스타일** — 개인 블로그, 취미

## 주요 기능 안내

### V-Audit (전문가 평가)
전문가에게 프로젝트 피드백을 받을 수 있어요.

- **평가 항목** (기본): 기획력, 완성도, 시장성, 독창성
- 프로젝트별로 평가 항목을 커스텀할 수 있어요
- 비밀 댓글로 솔직한 피드백도 가능
- 평가 결과는 프로젝트 소유자와 협업자만 볼 수 있어요
- 평가를 남기면 100포인트 적립!

### 투표 (스티커 폴링)
방문자가 프로젝트에 투표할 수 있어요.
- 기본 옵션: "당장 쓸게요!" / "조금 아쉬워요" / "더 연구해주세요"
- 커스텀 옵션도 설정 가능
- 투표 시 50포인트 적립

### 에피소드 (버전 관리)
프로젝트에 큰 업데이트가 있을 때 새 에피소드를 발행할 수 있어요.
- 이전 버전 자동 백업
- 변경 이력 타임라인으로 표시
- 팔로워에게 알림 자동 발송

### 협업자
팀 프로젝트의 경우 이메일로 협업자를 초대할 수 있어요.
- 협업자도 피드백/평가 열람 가능

### AI 도구
- **페르소나 빌더** (vibefolio.net/tools/persona) — 타겟 사용자 페르소나 생성
- **린캔버스** (vibefolio.net/tools/lean-canvas) — 사업 모델 캔버스 자동 생성
- **기회 매칭** (vibefolio.net/tools/opportunity) — 채용/공모전 매칭

## 마이페이지 기능

vibefolio.net/mypage 에서 다양한 관리 기능을 사용할 수 있어요.

- **내 프로젝트** — 게시한 프로젝트 관리, 수정, 삭제
- **평가 결과** — V-Audit 피드백 종합 리포트
- **컬렉션** — 저장한 프로젝트 모아보기
- **북마크** — 관심 프로젝트 저장
- **좋아요** — 좋아요 누른 프로젝트 목록
- **문의/제안** — 받은 제안 확인
- **프로필 설정** — 소개, 스킬, 관심사, 소셜 링크 관리

## FAQ

### 가입/이용 관련

**Q: 무료인가요?**
A: 네, 현재 모든 기능을 무료로 이용할 수 있어요.

**Q: 어떤 종류의 프로젝트를 올릴 수 있나요?**
A: 웹/앱, 디자인, 영상, 음악, 게임, 코드 등 모든 창작 프로젝트를 올릴 수 있어요.

**Q: GitHub 프로젝트도 올릴 수 있나요?**
A: 네! Quick Post에 GitHub URL을 넣으면 stars, 언어, README 등을 자동으로 수집해서 소개글을 만들어 줘요.

### Quick Post 관련

**Q: AI가 만든 소개글을 수정할 수 있나요?**
A: 물론이에요. AI가 초안을 만들어 주면, 게시 전에 자유롭게 수정할 수 있어요.

**Q: 썸네일이 자동으로 만들어지나요?**
A: 네. 사이트에 OG 이미지가 있으면 그걸 사용하고, 없으면 AI가 프로젝트에 맞는 썸네일을 자동 생성해요.

**Q: URL 분석 횟수에 제한이 있나요?**
A: 비로그인 시 하루 3회, 로그인 시 하루 10회까지 가능해요.

### V-Audit 관련

**Q: 평가는 공개되나요?**
A: 아니요. 평가 결과는 프로젝트 소유자와 협업자만 볼 수 있어요.

**Q: 평가 항목을 바꿀 수 있나요?**
A: 네, 기본 4개 항목(기획력, 완성도, 시장성, 독창성) 외에 최대 6개까지 커스텀 항목을 설정할 수 있어요.

### 기타

**Q: 채용 정보도 있나요?**
A: 네, vibefolio.net/recruit 에서 채용/공모전 정보를 확인할 수 있어요. 매일 자동으로 업데이트돼요.

**Q: 프로젝트 비공개가 가능한가요?**
A: 네, 게시 시 공개/비공개(unlisted) 설정이 가능해요.

**Q: 예약 게시가 되나요?**
A: 네, 원하는 날짜와 시간에 자동으로 게시되도록 예약할 수 있어요.

**Q: 문의는 어디로 하나요?**
A: vibefolio.net/contact 에서 문의하시거나, 이 채팅에서 바로 질문해 주세요!
